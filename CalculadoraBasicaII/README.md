# 🔢 Calculadora Básica II

Aplicativo Android desenvolvido para praticar conceitos avançados de desenvolvimento mobile, incluindo persistência de dados e autenticação.

---

## 📱 Descrição

Este projeto evoluiu de uma calculadora simples para um sistema completo com suporte a múltiplos usuários e histórico persistente, servindo como base para estudo de:

* **Autenticação:** Sistema de cadastro e login de usuários.
* **Persistência:** Uso de banco de dados SQLite para salvar o histórico de cálculos.
* **Relacionamentos:** Implementação de chaves estrangeiras para associar cálculos a usuários específicos.
* **Navegação:** Gerenciamento de múltiplas Activities e fluxo de dados com Intents.

---

## 🎯 Objetivos do projeto

* Praticar lógica de programação e manipulação de strings.
* Implementar o padrão DAO (Data Access Object) para persistência.
* Trabalhar com eventos de interface e atualização dinâmica de componentes.
* Entender a estrutura de banco de dados relacional no Android.

---

## 🛠️ Tecnologias

* Android Studio
* Java
* SQLite
* Gradle

---

## 📚 Conceitos praticados

* Componentes de UI (Buttons, TextView, EditText, etc.)
* Banco de Dados (SQLiteOpenHelper e CRUD)
* Chave Estrangeira (Foreign Keys) e Modelagem de Dados
* Ciclo de vida de Activities e Intents
* Organização de código (Packages: model, dao, database)

---

## ▶️ Como executar

1. Abra o projeto no Android Studio.
2. Aguarde o sync do Gradle.
3. Execute em um emulador ou dispositivo físico (API 24+).
