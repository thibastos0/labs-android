# 📱 Labs Android - Repositório de Estudos

Este repositório centraliza meus projetos de estudo e experimentações no desenvolvimento Android. O objetivo é explorar desde persistência local e lógica de UI até integrações complexas com serviços Google e Firebase.

---

## 📂 Projetos no Repositório

### 🌟 Destaques de Integração
*   **`AppTesteGoogleAuth/`**: Laboratório avançado focado na transição para a **Credential Manager API**, autenticação Google moderna, Firebase e Geolocalização (GPS + OpenStreetMap).
*   **`ProvaAndroid_Firebase/`**: Sistema de gestão acadêmica com autenticação híbrida (Google e E-mail), integrando **Cloud Firestore** e **Realtime Database** para sincronização em nuvem.

### 🛠️ Persistência e Lógica de Negócio
*   **`CalculadoraBasicaII/`**: App de calculadora evoluído com sistema de login, gerenciamento de múltiplos usuários e persistência de dados local utilizando **SQLite**.
*   **`ProvaAndroid/`**: Aplicação de cadastro de alunos e cálculo de médias. Explora o padrão **DAO** com **SQLite** nativo e possui implementações alternativas utilizando **Room Database**.

---

## 🎯 Objetivos de Aprendizado

*   **Autenticação**: Uso de Credential Manager, Google Sign-In e Firebase Auth.
*   **Backend & Nuvem**: Integração com Firebase (Firestore e Realtime Database).
*   **Persistência Local**: Trabalho com SQLite nativo e Room Persistence Library.
*   **Serviços de Localização**: Captura de coordenadas com `FusedLocationProviderClient` e integração com mapas via `osmdroid`.
*   **UI/UX**: Aplicação de Material Design 3, layouts responsivos e suporte a Edge-to-Edge.

---

## 🛠️ Tecnologias Utilizadas

*   **Linguagens**: Java (Principal) e Kotlin.
*   **Ferramentas**: Android Studio, Gradle (KTS), Git.
*   **APIs**: Google Play Services, Firebase SDK, Android Jetpack.

## 📌 Observação

Estes projetos servem como base de consulta técnica e registro de evolução no ecossistema Android, focando sempre em seguir as recomendações de segurança e arquitetura mais recentes do Google.
